안녕하세요!
휴먼인터페이스미디어 02분반 Team 1의 USER STUDY에 참여해 주셔서 감사합니다.

참여 안내

1. 환경 준비

  1. PC와 마우스를 준비해주세요.
  2. PC에서 해당 사이트에 접속한 뒤, 전체 화면 모드로 진행해주세요.

2. 게임 시작

  1. 문자로 안내받은 게임 모드를 선택하세요. (Mode 1, Mode 2)
  2. 본인의 이름을 입력한 후, Submit 버튼을 눌러주세요.
  3. Start Game 버튼을 누르면 게임이 시작됩니다.

3. 게임 진행

  1. N X N 블럭에 무작위로 불이 켜집니다. 순서와 위치를 기억해주세요.
  2. 화면에 ‘당신 차례입니다’ 문구가 나타나면, 기억한 순서대로 블럭을 클릭합니다.
  3. 블럭의 크기는 3 X 3, 4 X 4, 5 X 5로 변화하며, 각 크기에서 10단계를 진행합니다.
    • 단계가 올라갈수록 기억해야 할 블럭의 개수가 늘어납니다.
  4. 게임이 끝나면 전송 완료 메시지가 나타납니다. 반드시 메시지를 확인한 후 게임을 종료해주세요.
